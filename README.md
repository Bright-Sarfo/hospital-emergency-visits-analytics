# ğŸ’¡ Key Insights
- Average wait time: **35.3 minutes**
- Nearly **50% admission rate**, indicating high throughput pressure  
- Peak demand on **Saturdays, Thursdays, Sundays**  
- Busiest hours: **07:00, 11:00, 13:00, 23:00**  
- General Practice and Orthopedics dominate referrals

This means:
- Demand concentrated during specific days and hours
- Nearly equal split between admitted and non-admitted patients
- General Practice and Orthopedics drive referral volume
- Moderate satisfaction despite long wait times
